HELP

Generic

  • q — quit
  • <f1> — toggle help

Controls

  • <space> — switch view
  • f — toggle folding
  • a — sync current match
  • A — reset scroll syncing
  • <ctrl-n> — next file
  • <ctrl-p> — previous file

Layout

  • s — left/right split
  • v — top/bottom split
  • o — leave only current view visible
  • = — center split

Scrolling

  • <ctrl-e> — scroll down one line
  • <ctrl-y> — scroll up one line
  • t — move cursor to the top
  • z — center cursor
  • b — move cursor to the bottom

Navigation

  • g — go to the first line
  • G — go to the last line
  • j — navigate to the next change in current view
  • k — navigate to the previous change in current view
